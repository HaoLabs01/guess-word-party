# 猜词派对

一个适合聚会玩的中文猜词游戏。一个人手持手机不看屏幕，其他人看屏幕描述词语；猜对时手机向下翻，跳过时手机向上翻。

## 玩法

- 空格：开始游戏
- 向下翻或按 `ArrowDown`：猜对，得分 +1
- 向上翻或按 `ArrowUp`：跳过，不得分
- `Esc`：停止本局

## 功能

- 开始前选择词库和游戏时长：60 / 180 / 300 秒
- 多分类词库：成语、美食、动物、职业、运动等
- 前置摄像头录制描述者的动作和声音
- 本地运行时录像保存到 `recordings/`
- Vercel 线上版本支持录制后浏览器下载；云端持久保存录像需要另接对象存储

## 本地运行

```bash
node server.mjs
```

然后打开：

```text
http://127.0.0.1:8787/
```
