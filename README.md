# 猜词派对

一个适合聚会玩的中文猜词游戏。一名玩家负责猜词，其他玩家看屏幕上的词并描述，但描述时不能直接说出题面里出现的字。

## 玩法说明

1. 开始前选择词库和游戏时长，并在准备页完成动作权限和录制权限。
2. 猜词玩家不要看屏幕，其他玩家负责看屏幕上的词并描述。
3. 动作判断以“屏幕这一面”为准，不以手机背面为准：
   - 点头时，屏幕向下扣，表示猜对，得分 +1，并切换到下一个词。
   - 抬头时，屏幕向天翻，表示跳过，不得分，并切换到下一个词。
4. 点击开始后会先显示 5-4-3-2-1 倒计时，倒计时结束才开始计时、出词和录制。
5. 为减少误触，系统会先记录当前手持角度作为中立位；点头/抬头需要有明显角度变化，并连续保持后才会触发。触发后需要回到中立位，才能识别下一次动作。
6. 也可以用键盘操作：
   - `Space`：开始游戏
   - `ArrowDown`：猜对，得分 +1
   - `ArrowUp`：跳过，不得分
   - `Esc`：停止本局
7. 如果打开“录制视频”开关，前置摄像头画面会作为清晰的全屏背景显示；倒计时、词卡和按钮都会尽量透明，方便描述的人看到自己的动作。
8. 倒计时结束后自动录制，本局结束时自动停止录像；录像会使用更适合手机播放的小尺寸、低码率和分片写入，播放录像时会释放后台摄像头预览。

## 功能

- 开始前选择词库和游戏时长：60 / 180 / 300 秒
- 多分类词库：成语、美食、动物、职业、运动等
- 动作权限前置：开始游戏后不再弹出动作授权
- 录制视频 OFF / ON 开关：关闭时不请求摄像头录制，打开后本局自动录制
- 前置摄像头全屏预览，游戏中的倒计时、词卡、按钮和状态区都尽量透明，不遮挡描述者动作
- 录制描述者的动作和声音，结束后可在页面内播放或下载
- 本地运行时录像保存到 `recordings/`
- Safari 添加到主屏幕后，可以用首页右上角的“刷新”按钮重启应用
- Vercel 线上版本支持录制后浏览器下载；云端持久保存录像需要另接对象存储

## 本地运行

```bash
node server.mjs
```

```text
http://127.0.0.1:8787/
```
